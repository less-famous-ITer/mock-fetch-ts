Mock-fetch-ts 是一个用 `TypeScript` 编写的前端数据模拟器，针对的是原生的 fetch 函数。

文档地址: [mock-fetch-ts](https://www.yuque.com/u29138508/gki7q8/ofwzfpklbd1qwnyr?singleDoc)

## Feature

* 根据数据模板生成响应数据
* 提供对于原生 fetch 的数据模拟
